# 1101 Sequence Detector

This project implements a **1101 Sequence Detector** using a **Moore Finite State Machine (FSM)** in SystemVerilog. The design detects the binary sequence **1101** from a serial input stream and generates an output when the complete sequence is recognized.

- SystemVerilog design (`seq_detector.sv`)
- Testbench (`seq_detector_tb.sv`)
